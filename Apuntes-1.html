APUNTES SOBRE HTML, CSS Y JAVA SCRIPT
HTML (HyperText Markup Language)

HTML es el lenguaje de marcado que forma la estructura básica de las páginas web. Define elementos como encabezados, párrafos, enlaces, imágenes, listas, tablas, y formularios, entre otros. Es fundamental para construir la estructura semántica de los documentos web. HTML utiliza etiquetas para marcar diferentes partes del contenido, y cada etiqueta tiene un propósito específico (como h1 para encabezados o a para enlaces). HTML es independiente de los estilos visuales, que se suelen definir con CSS.

CSS (Cascading Style Sheets)

CSS se encarga de los estilos visuales en las páginas web, separando la estructura del contenido de su presentación. Permite definir colores, fuentes, márgenes, tamaños, posiciones y demás propiedades visuales, aplicándose en distintos niveles de la página (mediante hojas de estilo externas, internas, o en línea). Los estilos en CSS son jerárquicos y se heredan de elementos padre a hijos, aplicándose en cascada. CSS también facilita los Media Queries, que permiten que las páginas se adapten a diferentes tamaños de pantalla y dispositivos, un aspecto clave en el diseño web responsivo.

JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad y funcionalidad dinámica a las páginas web. Utilizado junto con HTML y CSS, JavaScript puede responder a eventos del usuario, modificar el contenido de la página de forma dinámica, realizar cálculos, y trabajar con APIs para manipular el contenido del DOM (Document Object Model). En la práctica, se recomienda minimizar el acceso al DOM para optimizar el rendimiento y aplicar las mejores prácticas de seguridad validando la entrada de datos para evitar posibles vulnerabilidades.

Los tres estándares se desarrollan bajo los principios de interoperabilidad y accesibilidad, lo que facilita que los desarrolladores creen experiencias consistentes en diferentes navegadores y dispositivos. La W3C mantiene estas tecnologías y las mejora continuamente para garantizar su relevancia y utilidad en el entorno digital actual.
